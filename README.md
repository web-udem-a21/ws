## Format de la remise

Chaque exercice devra être compris dans une balise `<section>` et précédé d'une balise `h1` contenant le titre de l'exercice.

```htmlembedded=
<!-- Exercice 1 --> 
<section> 
    <h1>Exercice 1</h1> 
    <form id="exercice-1"> 
    ... 
    </form> 
</section> 
```
